## 目录

每类题型按照子项目划分，每个项目下按照题目的难易程度氛围：easy, medium和hard三个包目录。

每道Leetcode题目注释符合如下规范，注释尽量描述清晰，方便后期回顾和总结。
```
/**
 * Title: name
 * 
 * 题目: 中文题目意思
 *
 * 思路: 解题目思路
 * 
 * 类似题型: 类型题型 
 *
 * @author name time
 */
```
